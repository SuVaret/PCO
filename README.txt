
PCOtesthmin V2.0
================

 This package implement the PCO method decribed in the preprint 
 "Numerical performance of Penalized Comparison to Overfitting for multivariate kernel density estimation"
 (https://hal.archives-ouvertes.fr/hal-02002275)


INSTALLATION
=============

** Before installation in R, type in a command shell
R CMD build /directory/of/folder/PCOtesthmin

** The previous command creates a .tar.gz file


** Then in an R session :
install.packages("/directory/of/PCOtesthmin.tar.gz/", repos=NULL, type="source")
